Dispatcher

Dispatcher.php : 23

Translate and dispatch a QUERY_STRING to the appropriate Controller or serve a valid cached version of the pages if one is available.

Accept queries formatted as follow : index.php?controller=ControllerName&action=ActionName&p1=v1&p2=v2&pn=vn"

Provide fallback routes for incomplete or junk queries.

note Dispatcher is not in charge of invalidating cache files. If a cache file exists and is not past its expiration date, it is considered valid and Dispatcher will serve it.
