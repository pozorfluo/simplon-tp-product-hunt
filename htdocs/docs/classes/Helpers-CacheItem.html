CacheItem

CacheItem.php : 21

Describe a cached item metadata.

Used in a Cache trove map.

note CacheItem does NOT hold the actual cached value/data, only metadata.

Actual cached value/data is stored in file named as follow : Cache::CACHE_PATH . CacheItem->filename . '.' .Cache->name
